About:

betahuhn.de bietet ein Formular an, mit dem Du eine Kurz-URL für eine bestehende Web-Adresse generieren kannst.

Wenn du dann auf die generierte Kurz-URL gehst, wirst du auf die ursprüngliche Seite weitergeleitet. Schau dir dazu die Datenschutzerklärung an.

Weiterleitung:

Die Weiterleitung zur ursprünglichen Website wird mithilfe von JavaScript ermöglicht, um zu verhindern das Suchmaschinen oder Twitter/Facebook Bots zugriff auf den Referrer haben. Außerdem wird durch die robots.txt Datei und des nofollow attributes versucht, dass betahuhn.de das Ranking der ursprünglichen Website nicht beeinflust.
